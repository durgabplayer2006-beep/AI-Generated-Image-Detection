# AI Generated Image Detection Using CNN

## Project Overview

This project detects whether an uploaded image is REAL or AI Generated using a Convolutional Neural Network (CNN).

The system is trained on the CIFAKE dataset and deployed using Flask to provide a user-friendly web interface for image classification.

## Features

- Upload image through web interface
- Detect REAL or AI Generated images
- Display prediction confidence
- Responsive and user-friendly design
- CNN-based image classification

## Technologies Used

- TensorFlow
- Keras
- Flask
- HTML
- CSS

- Pillow (PIL)
